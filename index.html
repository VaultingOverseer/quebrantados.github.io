Sobre o Jogo

Terra Arrasada é um jogo de escolhas ambientado em um futuro pós-apocalíptico. O jogador toma decisões que afetam a narrativa e o destino dos personagens.

O Projeto

Este projeto é parte de uma iniciativa escolar para desenvolver um jogo usando HTML, CSS e JavaScript, com foco em narrativas imersivas e decisões significativas.
